# burpsuite-noAuth

[README](README.md) | [README-CN](README-CN.md)

During the implementation of a penetration testing project, unauthorized access needs to be tested. Since there are a large number of API interfaces involved in the access process, manually deleting Cookies or tokens one by one is quite troublesome, so I wrote this plugin.
